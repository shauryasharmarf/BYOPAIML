# Injury Risk Prediction System (AI/ML Project)

## Overview
This project predicts injury risk using Machine Learning based on user inputs such as sleep, pain, intensity, and physical condition.

## Features
- Injury risk prediction (Low, Medium, High)
- BMI calculation and classification
- Injury location and cause input
- Doctor/physiotherapist recommendation
- Exercise advice

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Logistic Regression

## How to Run
1. Install dependencies:
   pip install pandas numpy scikit-learn

2. Run:
   python main.py

## Inputs
- Age, Weight, Height
- Sleep hours
- Pain level
- Workout intensity
- Previous injury
- Swelling & mobility issue
- Injury location & cause

## Outputs
- BMI + category
- Injury risk
- Recommendation
- Advice
